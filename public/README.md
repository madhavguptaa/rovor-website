# Public Assets Folder

This folder contains all static assets for the rovor website.

## Structure

- `/images` - Images used throughout the site
  - `/hero` - Hero section images
  - `/icons` - Icon files
  - `/logo` - Logo files
- `/fonts` - Custom font files (if any)

## Notes

- Add your hero image (woman taking selfie) to `/images/hero/`
- Add logo files to `/images/logo/`
- All icons are currently using inline SVG/emoji, but you can replace them with actual image files here
